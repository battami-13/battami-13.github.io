Password combinations (toy model)

This tool estimates how many different passwords are possible, based on length and the type of characters allowed. It uses simple maths (exponents) to show why longer passwords are harder to guess.

Hint: This is a simplified model. Real password strength also depends on patterns and attacker knowledge.

If each character in a password can be chosen from N symbols, and the password has L characters, then there are N^L possible passwords. For example, with 26 letters and length 4, there are 26^4 = 456,976 combinations.
